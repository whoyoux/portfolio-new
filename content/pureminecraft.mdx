## Project Description
PureMinecraft is a comprehensive platform for managing Minecraft server access and subscriptions. The application provides a seamless experience for users to purchase server access, manage their subscriptions, and connect their Minecraft accounts.

## Key Features
- Secure payment processing for server access
- User authentication and account management
- Subscription management system
- Automated email notifications
- Minecraft nickname verification
- User dashboard for account control
- Account deletion functionality
