## Project Description
A full-featured e-commerce platform built with modern web technologies. This project demonstrates a complete online shopping experience with product management, shopping cart functionality, and secure payment processing.

## Key Features
- Product catalog with categories and search
- Shopping cart and wishlist functionality
- User authentication and profile management
- Secure payment processing
- Order tracking and history
- Admin dashboard for product management
- Responsive design for all devices
